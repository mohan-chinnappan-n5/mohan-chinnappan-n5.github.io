Git Cheat Sheet

Common Git commands and their usage.

Command Description Example
`git init` Initialize a new Git repository. `git init my-repo`
`git clone ` Clone a repository into a new directory. `git clone https://github.com/user/repo.git`
`git status` Show the working tree status. `git status`
`git add ` Add a file to the staging area. `git add README.md`
`git commit -m ""` Commit changes with a message. `git commit -m "Add README file"`
`git push` Push changes to the remote repository. `git push origin main`
`git pull` Fetch and merge changes from the remote repository. `git pull origin main`
`git branch` List all branches in the repository. `git branch`
`git checkout ` Switch to a specified branch. `git checkout feature-branch`
`git merge ` Merge a specified branch into the current branch. `git merge feature-branch`
`git log` Show the commit history. `git log`
`git reset ` Reset current HEAD to the specified state. `git reset HEAD~1`
`git revert ` Create a new commit that undoes the specified commit. `git revert abcd1234`
`git stash` Stash the changes in a dirty working directory. `git stash`
`git stash pop` Apply the stashed changes and remove them from the stash. `git stash pop`

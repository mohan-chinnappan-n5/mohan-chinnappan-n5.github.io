Common Salesforce Apex commands and their usage.

Command Description Example
`System.debug` Logs debug information for troubleshooting. `System.debug('Hello, Apex!');`
`DML: insert` Inserts a new record into Salesforce. `insert new Account(Name='Test Account');`
`DML: update` Updates an existing record. `update accountRecord;`
`DML: delete` Deletes a record from Salesforce. `delete accountRecord;`
`SOQL Query` Query Salesforce records. `[SELECT Id, Name FROM Account];`
`SOSL Query` Search across Salesforce records. `FIND 'test' IN ALL FIELDS RETURNING Account(Name);`
`Try-Catch` Handles exceptions and errors. 
try { /* Code */ } catch (Exception e) { System.debug(e); }
`Future Method` Runs code asynchronously. `@future public static void myFutureMethod() { /* Code */ }`
`@InvocableMethod` Expose methods to Process Builder/Flow. `@InvocableMethod public static void myMethod(List<String> ids) { /* Code */ }`
`Batch Apex` Processes large amounts of data asynchronously. `public class MyBatch implements Database.Batchable<sObject> { /* Code */ }`
`Test.startTest / Test.stopTest` Defines a test block in Apex tests. 
Test.startTest(); /* Code */ Test.stopTest();
`System.assertEquals` Asserts equality in tests. `System.assertEquals(expected, actual);`

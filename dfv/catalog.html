Help for the viewer - URL arguments

  • d : data file in this server or url - supported types: csv, psv, json
  • x : Column number (zero indexed) to be used for the chart (default : 0 )
  • cw: Chart width (default: 400)
  • ch: Chart height (default: 300)
  • ct: Chart type (default: bar, you can use: bar, line, area )
  • c: Chart needed? (default:1, use 0 to disable chart)
  • cc: Chart Dim needed? (default:0, use 1 to enable color dim chart)
  • cf: Chart field (default:1)
  • wc: World Cloud Needed (default:1, use 0 to disable World Cloud)

How to Use

Follow these steps to start using Apex Code Assistant:

üîπ Step 1: Set Up Your Local LLM with Ollama

Run a local Ollama server to power the extension. Use these commands:

export OLLAMA_ORIGINS=chrome-extension://*
ollama serve

üîπ Step 2: Check Available Models

List the models available on your local Ollama server:

gemma3:latest              a2af6cc3eb7f    3.3 GB    5 hours ago      
tinyllama:latest           2644915ede35    637 MB    13 months ago

üîπ Step 3: Install and Use the Extension

Install the extension, paste your Apex code, select a model, and generate documentation or test classes.

Use the "Copy to Clipboard" or "Download" buttons to save your generated Apex code.

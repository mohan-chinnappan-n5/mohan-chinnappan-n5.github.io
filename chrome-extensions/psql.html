How to Use

Follow these steps to set up and use the PostgreSQL SQL Editor:

üîπ Step 1: Set Up the Backend Server

Clone the repository and install dependencies for the Node.js backend:

curl -O https://raw.githubusercontent.com/mohan-chinnappan-n5/backend-servers/refs/heads/main/server.js
npm install

Configure your PostgreSQL connection in config.json or a .env file, then start the server:

node server.js 3000

üîπ Step 2: Access the Editor

Open index.html in a browser, set the server URL in the Settings modal (e.g., http://localhost:3000/), and start writing queries:

SELECT * FROM employees

Use the table dropdown for autocompletion and export results as CSV.

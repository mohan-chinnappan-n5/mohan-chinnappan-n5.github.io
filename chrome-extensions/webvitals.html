How to Use

Follow these simple steps to start measuring Web Vitals with our extension:

üîπ Step 1: Install the Extension

Add the Web Vitals extension to Chrome from the Chrome Web Store.

Click the "Install Now" button in the navbar or visit the Chrome Web Store to install.

üîπ Step 2: Navigate to a Web Page

Open any web page (http or https) in Chrome where you want to measure performance.

The extension uses the `activeTab` permission to access the current tab‚Äôs URL and communicate with the content script to fetch Web Vitals metrics, ensuring real-time performance data for the active page.

üîπ Step 3: Open the Popup

Click the Web Vitals extension icon in the Chrome toolbar to view the metrics.

The popup displays metrics like LCP, INP, and CLS, with color-coded indicators and detailed analysis (e.g., LCP element, worst CLS cluster).

üîπ Step 4: Interact for INP and FID

Interact with the page (e.g., click a button or type in a form) to measure INP and FID metrics.

The extension uses host permissions `http://*/*` and `https://*/*` to inject the content script into web pages, enabling the collection of Web Vitals metrics while excluding restricted URLs for security.

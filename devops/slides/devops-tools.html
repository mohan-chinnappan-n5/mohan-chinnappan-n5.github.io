## Key Concepts First prepare artifacts [Delta Deployment](https://mohan-chinnappan-n5.github.io/sfbooks/sfdevnotes/deployments/delta.html) ``` - Create a folder with this package.xml and destructiveChanges.xml - Upload the package.xml in this app - Fill up the required values in this form - Store the downloaded script (bundle-script.sh) in that folder - Run this script: ```bash bundle-script.sh > results.txt``` in that folder - Any errors in the script run will be rendered on the screen - Script prepares a zip bundle for the deployment and also the README.md - README.md explains how with one single command you can deploy the bundle ``` The Bundler App is available [here](https://mohan-chinnappan-n5.github.io/sf/pkg-bundle/app.html) [![Bundler](img/bundler-1.png)](img/bundler-1.png)

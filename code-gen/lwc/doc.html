Overview

LWC CodeGen is a web-based tool designed to simplify the creation of Salesforce Lightning Web Components (LWC). By inputting a valid SOQL query, users can generate LWC code using predefined templates such as uiRecordApi, ApexWire, ApexREST, and TCRM. The app features a split-pane interface with Monaco Editors for input and output, offering syntax highlighting and real-time feedback.

Key features include:

  • SOQL parsing with customizable RegEx
  • Multiple LWC templates
  • Monaco Editor integration for code editing
  • Copy-to-clipboard and file download options

Usage

Getting Started

1. Open the app in your browser.

2. In the left pane, enter a valid SOQL query in the SOQL Query editor (e.g., SELECT Id, Name FROM Account).

3. Adjust the RegEx for SOQL Parsing if needed (default: SELECT\s+(.*?)\s*FROM\s*([a-zA-Z_$0-9]*)).

4. Select a template from the Select Template dropdown.

5. View the parsed RegEx results in the RegEx Results editor.

6. Check the generated code in the right pane, organized in tabs.

7. Use the Copy to Clipboard button or download files individually or all at once.

Tips

  • The SOQL Query editor supports SQL syntax highlighting.
  • The RegEx Results editor is read-only and displays JSON output.
  • Downloads are individual files (ZIP functionality requires additional libraries).

Templates

LWC CodeGen supports the following templates:

uiRecordApi

Generates an LWC using the uiRecordApi to fetch record data. Outputs:

  • .html: Displays fields in a card layout
  • .js: Wires record data with getters
  • .js-meta.xml: Metadata for deployment

ApexWire

Generates an LWC with Apex controller integration using the wire service. Outputs:

  • .cls: Apex controller
  • .cls-meta.xml: Apex metadata
  • .html: Datatable for records
  • .js: Wires Apex method
  • .js-meta.xml: Metadata

ApexREST

Generates an Apex REST controller. Outputs:

  • .cls: REST endpoint

TCRM

Generates an LWC for Tableau CRM (TCRM) dashboards. Outputs:

  • dashboard.json: Dashboard step
  • .html: Simple display
  • .js: Component logic
  • .js-meta.xml: Metadata

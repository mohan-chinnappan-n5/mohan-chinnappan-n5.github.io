Key Features

Real-Time EPT Tracking

Instantly view EPT and Total EPT values for any active Salesforce page hosted on force.com domains.

User-Friendly Popup

Access a clean interface with color-coded metrics (green for normal, red for errors).

One-Click Refresh

Reload the active tab and update EPT metrics with a single click to troubleshoot performance issues.

Secure & Lightweight

Built with Manifest V3, ensuring security and efficiency on force.com domains.

NOTE: This extension does not collect any user data.

How to Use

  1. Install the Salesforce EPT Monitor extension from the Chrome Web Store.
  2. Navigate to any Salesforce Lightning page on a force.com domain.
  3. Click the extension icon to view live EPT and Total EPT metrics in the popup.
  4. Use the refresh feature to re-evaluate performance after making changes.

# Sample Markdown Content

## Overview
This is a sample description to demonstrate the rendering of Markdown content in the left pane of the application.

## Features
- Dynamically render Markdown files in the browser.
- Use Mermaid.js for creating diagrams.
- Support for both Markdown and Mermaid diagrams simultaneously.

Quick Start (60‑second method)

Programs A, B, C are separate schedules. If all start times in Program B are set to OFF, Program B will never run — even if run times and days are set.

  1. Turn the dial to Start Times.
  2. Press PRG until you see B on the display.
  3. Use – to change each start time to OFF. Repeat for all 4 start slots.
  4. Return the dial to Run.

That’s it. No start time = no watering for Program B.

Controller Buttons You’ll Use

  • PRG — cycles A → B → C
  • + / – — change values
  • Dial positions: Start Times, Water Days, Run
If you see unexpected extra watering, you probably have start times in more than one program (A and B). Clear Program B to fix it.

Option 1 — Clear all start times for Program B (recommended)

1
Turn the dial to Start Times.
2
Press PRG until the screen shows program B.
3
Use +/– to set the first start time to OFF.
4
Press the ▶ or next arrow (if available) to move to start time 2. Repeat until all four start times show OFF.
5
Return dial to Run. Program B won’t run anymore.
Only start times launch a program. Run times per station and water days do nothing without at least one start time.

Option 2 — Disable watering days for Program B

This is a second way to stop Program B. With no active days, the program never runs.

1
Turn dial to Water Days.
2
Press PRG to select B.
3
Use +/– to set all days to OFF (or deselect every day symbol).
4
Return dial to Run.
If someone later re‑enables any day and there’s still a start time set, Program B will run again. Option 1 is safer.

Option 3 — Reset/Erase Program B (optional)

Use if you want Program B completely cleared.

1
Press and hold PRG while on Program B until values clear/reset.
2
Verify: Start Times all show OFF, and Water Days are empty.
Avoid factory‑resetting the entire controller unless necessary — it wipes all programs.

Why did my yard water twice?

You likely have start times in both Program A and Program B. Clear start times in Program B (Option 1).

How do I switch between programs?

Press PRG to cycle A → B → C. The current program letter shows on the screen.

What if I can’t find the OFF value?

While on a start time, hold – to scroll quickly until the display shows OFF.

Do I need to change run times per station?

No. Run times don’t matter if there are no start times. Clearing start times is enough.

Can I use only Program A?

Yes. Many homeowners keep everything on Program A and leave B/C empty. Use B/C only for special schedules.

My model looks slightly different

Pro‑C panels vary by revision, but the navigation (dial → PRG → +/–) is consistent. The same steps apply.

Reference — Button sequence cheat‑sheet

Clear all start times (Program B)
  1. Dial → Start Times
  2. PRG → B
  3. Start 1: – until OFF
  4. Next start → repeat for Starts 2–4
  5. Dial → Run
Disable Water Days (Program B)
  1. Dial → Water Days
  2. PRG → B
  3. Turn off every day
  4. Dial → Run
Full Program B reset
  1. PRG (hold) on B
  2. Verify Start Times show OFF
Pro tip: Label a small sticker inside the controller door — “Program B = OFF (clear start times)”. Future‑you will thank you.

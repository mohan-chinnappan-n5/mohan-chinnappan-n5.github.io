Overview

The Hunter Pro‑C is a reliable modular irrigation controller used for residential and commercial systems. It supports multiple programs (A, B, C), each with independent start times, run times, and watering days. This manual focuses on safely turning off Program B without affecting Program A or C.

Safety & Precautions

  • Always turn the dial to Run or remove power before handling wiring.
  • Do not perform electrical wiring unless you are comfortable with low-voltage (24 VAC) systems. If unsure, contact a licensed irrigation technician or electrician.
  • Factory resets will erase all programs. Use them only if intended.

Controller Panel & Controls

Common buttons and dial positions:

  • PRG — Cycle Program A → B → C
  • Start Times (dial) — Set program start times
  • Water Days (dial) — Select watering days
  • Run — Normal operating position
  • +/– — Increase / decrease values
  • Station zone selection and run time settings (usually labeled Run Time or Station)

Tip: The display shows which program letter (A/B/C) is active. Always confirm you are on Program B before changing settings.

Quick Start — 60‑second method

  1. Turn the dial to Start Times.
  2. Press PRG until the display shows B.
  3. Use – to change each start time to OFF (repeat for all start slots).
  4. Return dial to Run. Program B will not run.

Complete Methods to Disable Program B

Method 1 — Clear Start Times (Recommended)

  1. Turn dial to Start Times.
  2. Press PRG until B is displayed.
  3. For each start slot (1–4): select the start time and press – until the display shows OFF.
  4. Verify all start times read OFF.
  5. Return dial to Run. Confirm no scheduled runs from Program B occur.

Why recommended: With no start times, the program cannot trigger, regardless of station run times or day settings.

Method 2 — Disable Water Days

  1. Turn dial to Water Days.
  2. Press PRG until B is displayed.
  3. Use +/– to deselect every day (so no days are highlighted).
  4. Return dial to Run.

Caveat: If someone later re-enables a day but start times remain, Program B can run again. Combine with Method 1 for safety.

Method 3 — Reset/Erase Program B

  1. Press and hold PRG while the controller shows Program B — some revisions clear values for that program.
  2. Check both Start Times and Water Days to ensure they are cleared.

Do not perform a full factory reset of the controller unless you intend to erase all programs (A, B, and C).

Method 4 — Advanced: Use Sensor / Master Valve Settings

As an advanced override, you can assign a rain sensor or master valve to disable a program. This is useful for temporary pauses (like contractor work):

  1. Identify the controller menu for Sensor or Master Valve.
  2. Set the controller to require sensor to be active for Program B (depends on firmware).
  3. Activate the sensor (or set it to "triggered") to temporarily prevent Program B from running.

Note: Menu names vary; consult the model-specific manual if your Pro‑C firmware differs.

Practical Examples & Use Cases

Example 1 — I only want Program A to run

  1. Clear all start times in Program B and C (Method 1).
  2. Confirm Program A has the desired start time(s) and run times per station.

Example 2 — Seasonal Pause for Program B (e.g., Winter)

  1. Set all water days for Program B to OFF (Method 2).
  2. Optionally annotate the controller with a sticker noting the pause start/end dates.

Example 3 — Temporary contractor pause

  1. Use a physical jumper on the sensor terminals or set sensor to triggered to stop all programs, or clear B start times for a targeted pause.

Wiring, Zones & Fuses

The Pro‑C uses low‑voltage 24 VAC from a transformer to power valves. Typical terminal layout:

TerminalFunction
24VAC / TRANSTransformer input (24 VAC)
COMCommon for valve solenoids
1,2,3... (station terminals)Valve outputs for irrigation zones
MV / PumpMaster valve or pump start relay
SensorRain sensor input (optional)

Wiring tips:

  • Turn off power to the transformer before connecting or disconnecting valve wires.
  • Label wires at both ends for easier identification.
  • Use watertight connectors inside valve boxes for outdoor terminals.

Maintenance & Battery

  • Many Pro‑C units have a battery/back‑up for timekeeping. Replace batteries yearly to avoid schedule loss during power outages.
  • Inspect wiring and terminal screws annually for corrosion.
  • Clean controller enclosure to avoid dirt and moisture buildup.

Troubleshooting

Problem: Program B still runs after I cleared start times

  • Check Program A and C — they may also have start times that run the same zones.
  • Verify you actually saved the OFF value for each start slot (some revisions require pressing another button to confirm).
  • Confirm the dial is in Run and not in a manual watering mode.

Problem: Zones run unexpectedly

  • Check if a manual start (rain sensor override) is active.
  • Confirm that wiring for zone valves is correct and that no external relay is activating them.

FAQ

Can I permanently delete Program B?

You cannot remove the B program slot, but you can clear its values so it does nothing. Keep documentation (sticker) on the controller to remind others.

Will clearing Program B affect Program A?

No — programs are independent. Clearing B has no effect on A or C.

How do I backup my controller settings?

Older Pro‑C models don’t support digital backups. Document settings manually: program start times, run times per station, and watering days. Some smart controllers paired with apps allow cloud backups (not applicable to standalone Pro‑C without module).
